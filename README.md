# 🏋️ Spor Salonu Yönetim Sistemi

A web-based fitness management system that provides workout plans, diet programs, body type-based training, and membership features.

---

## 🚀 Proje Hakkında

Bu proje, kullanıcıların spor, beslenme ve antrenman süreçlerini tek bir platform üzerinden yönetebilmesi için geliştirilmiştir. Amacı, kişiye özel fitness deneyimi sunmaktır.

---

## ✨ Özellikler

- 🧍 Vücut tipine göre antrenman programları
- 🥗 Diyet ve beslenme planları
- 📅 Haftalık egzersiz programı
- 🧑‍🏫 Eğitmen profilleri
- 👤 Üyelik ve kullanıcı paneli
- 📊 (Planlanan) ilerleme takibi sistemi

---

## 🛠️ Kullanılan Teknolojiler

- HTML5
- CSS3
- JavaScript
- (Opsiyonel) React / Next.js
- (Opsiyonel) Node.js
- (Opsiyonel) MongoDB / Firebase

---

## 🎯 Amaç

Modern bir fitness platformu oluşturarak kullanıcıların antrenman, diyet ve gelişim süreçlerini dijital olarak yönetmesini sağlamak.
